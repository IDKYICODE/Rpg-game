# RPG Game

A role-playing game built with Unity and C#.

## Requirements

- Unity (LTS version recommended)
- .NET / C#

## Getting Started

1. Clone the repo
2. Open Unity Hub
3. Click **Add project from disk** and select the repo root
4. Let Unity import assets
5. Press **Play** in the Unity Editor

## Build

`File > Build Settings` — select target platform and click **Build**.
